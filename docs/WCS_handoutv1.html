WCS Basketball Website Overview

Project Purpose

The new WCS Basketball Club website was developed to modernize how the club communicates, organizes, and presents itself to families and the wider community. It provides a unified digital home for players, parents, and coaches — reflecting the club’s 25-year legacy while positioning WCS for future growth.

Key Features

Public-Facing Features
  • Home Page: Highlights club identity with a hero banner, fan zone links, recent news, and upcoming features.
  • About Page: Shares WCS’s history, mission, and values for families and potential sponsors.
  • Teams Page: Displays all active teams dynamically.
  • Team Detail Pages: Dedicated team pages showing coaches, updates, and schedules.
  • Schedule Page: Central hub for game, practice, and event schedules accessible to parents on any device.
  • Practice Drills Library: Curated collection of drills to support WCS and guest coaches.
Coaches Dashboard
  • Secure login for authorized coaches.
  • Update team schedules, news, and drills directly from the dashboard.
  • Instant synchronization to public pages for parents and players.
Hosting & Access
  • Deployed on Vercel with WCS’s existing domain.
  • Responsive design — optimized for desktop, tablet, and mobile.
  • Functions as a Progressive Web App (PWA) — can be “installed” on phones like a native app.

Technical Overview

  • Framework: Next.js (React-based Progressive Web App)
  • Hosting: Vercel Cloud Platform
  • Database / API: Integrated backend endpoints for teams, news, and schedules
  • UI / Styling: Tailwind CSS and component-based design for scalability
  • Authentication: Secure coach login via session-based system

PWA vs. Native Mobile App

Progressive Web App (PWA): Works through any web browser but can be installed on a phone’s home screen. It updates automatically, costs less to maintain, and provides offline caching and push notifications.

Native Mobile App: Requires App Store / Google Play submission and separate development for iOS and Android. Offers deeper phone integration but is more expensive to build and maintain.

For WCS: A PWA offers 90% of the benefits of a native app at a fraction of the cost — ideal for a youth sports organization focused on accessibility and sustainability.

Future Opportunities

  • Online Shop: Sell team and club gear to generate revenue.
  • Sponsorship Integration: Display sponsor logos and links to support club funding.
  • Scholarship Program: Channel sponsorship funds to help families with financial barriers.
  • Push Notifications: Real-time updates for weather cancellations or game reminders.
  • Community Additions: Photo gallery, highlights, or volunteer sign-up areas.

Summary

The WCS Basketball Website provides a foundation for stronger communication, engagement, and growth. It honors the club’s 25-year tradition while giving parents, coaches, and players an easy, reliable tool to stay connected — wherever they are.
